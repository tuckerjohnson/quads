Usage

Required Fonts:

- haydn (https://github.com/OpenLilyPondFonts/haydn)
- Junicode
- Syne

From a command prompt in the root directory:

cat variables/* eng-files/* front-matter/* book.ly | lilypond -dno-point-and-click --output="quads" -
